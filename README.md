# 🍰 نظام إدارة المخبز - موقع ويب كامل

نظام إدارة المخبز المتقدم بالذكاء الاصطناعي مع واجهة ويب جميلة وسهلة الاستخدام

## 🌟 المميزات الجديدة

### 🎨 **واجهة ويب جميلة**

- تصميم مودرن ومتجاوب
- دعم اللغة العربية من اليمين إلى اليسار
- ألوان وأيقونات جذابة
- تحريك سلس للانتقالات

### 📊 **لوحة تحكم شاملة**

- إحصائيات مباشرة
- عرض عدد المستخدمين والمنتجات
- معلومات الطلبات والمتاجر
- تحديث تلقائي للأرقام

### 🛠️ **إدارة متقدمة**

- إدارة المنتجات بالكامل
- تتبع الطلبات والمبيعات
- تقارير وتحليلات مفصلة
- مساعد ذكي للاستفسارات

## 🚀 التشغيل السريع

### 1. التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/yourusername/bakery-management-system.git
cd bakery-management-system

# تثبيت المتطلبات
pip install -r requirements.txt
```

### 2. التشغيل

```bash
# تشغيل الخادم
python main.py
```

### 3. الوصول للموقع

افتح المتصفح واذهب إلى:

```
http://localhost:8000
```

## 📱 واجهة المستخدم

### 🏠 **الصفحة الرئيسية**

- عرض إحصائيات شاملة
- نظرة عامة على النشاط
- أزرار سريعة للوصول لكل قسم

### 📦 **إدارة المنتجات**

- عرض جميع المنتجات في جدول
- معلومات السعر والكمية
- حالة المنتج (متوفر/غير متوفر)
- تحديث مباشر للبيانات

### 🛒 **إدارة الطلبات**

- قائمة الطلبات الحالية
- تفاصيل العملاء
- تتبع حالة الطلبات
- معلومات المبالغ

### 📊 **التحليلات**

- تقارير مفصلة بالذكاء الاصطناعي
- تحليل الأداء والمبيعات
- إحصائيات متقدمة
- رسوم بيانية (قريباً)

### 🤖 **المساعد الذكي**

- واجهة دردشة جميلة
- استفسارات باللغة العربية
- إجابات ذكية عن البيانات
- تحليل فوري للطلبات

## 🎯 API المتاحة

| نوع الطلب | العنوان          | الوصف                  |
| --------- | ---------------- | ---------------------- |
| `GET`     | `/`              | الصفحة الرئيسية        |
| `GET`     | `/api/health`    | فحص حالة النظام        |
| `POST`    | `/api/chat`      | الدردشة مع AI          |
| `GET`     | `/api/stats`     | إحصائيات سريعة         |
| `POST`    | `/api/query`     | استعلام قاعدة البيانات |
| `GET`     | `/api/analytics` | التحليلات الذكية       |

## 🔧 الإعدادات

### إعداد قاعدة البيانات

```python
# في env_config.py
DATABASE_URL = "mysql://username:password@host:port/database"
```

### إعداد الذكاء الاصطناعي

```python
# في env_config.py
GEMINI_API_KEY = "your-gemini-api-key"
GEMINI_MODEL = "gemini-1.5-pro-latest"
```

## 🌐 النشر على PythonAnywhere

### خطوات النشر:

1. رفع الملفات إلى PythonAnywhere
2. إنشاء Web App جديد
3. تحديد مسار WSGI: `wsgi.py`
4. إعداد Static Files: `/static/`
5. تحديث متغيرات البيئة
6. إعادة تشغيل الموقع

للتفاصيل الكاملة، راجع [دليل النشر](DEPLOYMENT_GUIDE.md)

## 📂 هيكل المشروع

```
bakery-management-system/
├── templates/              # قوالب HTML
│   └── index.html          # الصفحة الرئيسية
├── static/                 # الملفات الثابتة
│   ├── css/
│   │   └── style.css       # أنماط التصميم
│   ├── js/
│   │   └── app.js          # برمجة JavaScript
│   └── images/             # الصور
├── api/                    # واجهات API
│   ├── web_routes.py       # مسارات الويب
│   ├── routes.py           # مسارات API
│   └── models.py           # نماذج البيانات
├── ai/                     # خدمات الذكاء الاصطناعي
├── database/               # إدارة قاعدة البيانات
├── config.py               # إعدادات النظام
├── main.py                 # نقطة البداية
├── web_server.py           # خادم الويب
├── wsgi.py                 # إعداد WSGI
└── requirements.txt        # المتطلبات
```
